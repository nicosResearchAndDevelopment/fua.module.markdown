## Escaping Characters

| Character | Name                                           |
|-----------|------------------------------------------------|
| \\        | backslash                                      |
| \`        | backtick (see also escaping backticks in code) |
| \*        | asterisk                                       |
| \_        | underscore                                     |
| \{ \}     | curly braces                                   |
| \[ \]     | brackets                                       |
| \< \>     | angle brackets                                 |
| \( \)     | parentheses                                    |
| \#        | pound sign                                     |
| \+        | plus sign                                      |
| \-        | minus sign (hyphen)                            |
| \.        | dot                                            |
| \!        | exclamation mark                               |
| &#124;    | pipe (see also escaping pipe in tables)        |
